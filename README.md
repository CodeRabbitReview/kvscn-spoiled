# Storage

Simple key-value storage to save different types of data

## Tests
#### To run tests run from console write:
```bash
make test
```

## Installation

Use the go get to install storage in your machine.

```bash
go get github.com/mishaprokop4ik/storage
```

## Usage

Go to storage folder

```bash
go run main.go
```

## Make operations
Run tests
```bash
make test
```
make golint
```bash
make lint
```
fmt project
```bash
make fmt
```
Pre commit operations: fmt lint and test operations
```bash
make precommit
```
build binary file of project
```bash
make build
```
